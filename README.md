# Lexer with Flex and C

Implementation of a simple Lexer that recognizes the following tokens:
- real numbers
- operators
- comments in c++
- comments in python/ruby
- strings
- parenthesis
- brackets
- curly brackets
- identifiers
- semicolon
- colon
- coma
- library in c++ and python

Using Flex and C and then output the results to an html file
